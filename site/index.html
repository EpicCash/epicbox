713.grinbox

A transaction building service for Grin, currently under development on Github.

Grin is a cryptocurrency with a focus on privacy and scalability. Transactions are interactive, requiring the Sender and Recipient to interact over a single round trip in order to build a valid transaction.

What's 713.grinbox?

713.grinbox facilitates transaction building by acting as a relaying service that routes messages back and forth between senders and recipients.

Current status

Integrated and operational in wallet713 - a wallet that aims to make it easy to store, send, and swap Grin. Build the latest version and try it out.
